# obs_studio_moxfield_spelltable_animated_ui_example
An export of the OBS Studio configuration, including images and video animations, for running Moxfield as an interface through a virtual camera on Spelltable.

![Demo](demo.gif)

To import it, open OBS Studio, go to 'Scene Collection', and select 'Import...' to load export.json.
Some elements may require minor adjustments for alignment or personal preference, and it should work correctly in most cases.
